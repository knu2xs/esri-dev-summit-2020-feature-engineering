What is a Feature?

GIS

A representation of a real-world object on a map. - Esri GIS Dictionary

Data Science

A machine learning feature is an individual measurable property or characteristic of a phenomenon being observed. The concept of feature is related to that of explanatory variable. - Wikipedia

Feature Engineering

Feature engineering is the process of using domain knowledge to extract features from raw data via data mining techniques...to improve the performance of machine learning algorithms. - Wikipedia

Using GIS and data science tools and methodologies to create quantitative measures of geographic relationships and phenomena used for machine learning model development.
